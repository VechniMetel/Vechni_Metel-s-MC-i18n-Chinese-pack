# ME输入输出总线

<GameScene zoom="8" background="transparent">
  <ImportStructure src="../structure/cable_import_export_bus.snbt"></ImportStructure>
</GameScene>

ME输入输出总线是<ItemLink id="ae2:import_bus" />与<ItemLink id="ae2:export_bus" />的功能结合体。通过配置过滤器：
- 符合过滤条件的物品堆将执行输出操作
- 未在过滤器中的物品堆将执行输入操作
