# 熵变机械臂

<ItemImage id="entropy_manipulator" scale="4" />

该工具可通过右键点击加热物体，Shift+右键点击冷却物体。主要功能包括：
- 蒸发/冻结水体
- 将岩浆固化为黑曜石
- 将原木烧制成木炭
- 将圆石烧制成石头

若无特定操作对象时，可作为打火石使用。

能量可通过<ItemLink id="charger" />充能器补充。
