# ME空间塔

<BlockImage id="spatial_pylon" p:powered_on="true" scale="8" />

空间塔用于[空间存储操作](../ae2-mechanics/spatial-io.md)中生成空间力场并划定存储区域范围。

* 每段连续的塔链消耗1个[频道](../ae2-mechanics/channels.md)
* 塔链长度需至少2个方块方可生效
