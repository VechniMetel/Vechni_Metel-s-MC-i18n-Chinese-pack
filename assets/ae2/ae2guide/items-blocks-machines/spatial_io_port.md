# ME空间IO端口

<BlockImage id="spatial_io_port" p:powered="true" scale="8" />

空间IO端口用于[空间存储操作](../ae2-mechanics/spatial-io.md)，可容纳[空间存储元件](spatial_cells.md)并控制空间交换过程。

该端口支持通过漏斗、AE2总线等物品传输设备实现自动化存取元件操作。
