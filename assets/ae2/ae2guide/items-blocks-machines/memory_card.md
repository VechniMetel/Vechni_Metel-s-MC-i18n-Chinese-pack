# 内存卡

<ItemImage id="memory_card" scale="4" />

内存卡用于在AE2[设备](../ae2-mechanics/devices.md)间复制粘贴配置、链接[P2P通道](p2p_tunnels.md)，并能将[升级卡](upgrade_cards.md)批量安装至设备。

- **Shift+右键点击**：复制设备配置或生成新的P2P链接频率
- **右键点击**：粘贴配置/升级卡或绑定P2P频率
